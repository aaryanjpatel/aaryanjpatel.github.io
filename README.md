# Portfolio Site

A simple portfolio website using Jekyll, HTML, CSS, and Markdown.

## How to Use

1. Edit the Markdown files in the `content/` folder:
   - `about.md`: Your about section
   - `projects.md`: Your projects
   - `contact.md`: Contact information

2. Customize `style.css` for styling.

3. To test locally, install Jekyll and run `jekyll serve`.

## Deploy to GitHub Pages

1. Push this repo to GitHub.
2. Enable GitHub Pages in the repo settings (select main branch).
3. GitHub will build the site using Jekyll and host it at `https://yourusername.github.io/repository-name/`
